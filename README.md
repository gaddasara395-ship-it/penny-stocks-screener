# Penny Stocks Screener

## Overview
This project is a Python-based Penny Stocks Screener that helps retail investors
identify potential high-growth penny stocks using market data.

## Problem Statement
Retail investors often struggle to identify promising penny stocks due to high
volatility and lack of analytical tools.

## Solution
The screener filters NSE-listed stocks using:
- Current price
- Average trading volume
- One-month return

## Technologies Used
- Python
- yfinance
- pandas
- numpy
